# THE SITE

### 3D Artist • Programmer • Blender & Unreal Enthusiast  

A personal website built to showcase my work in **3D art, design, and programming**.  
It blends **artistic visual style** with **clean code**, featuring animated UI, liquid-glass effects, and a responsive layout powered by **Tailwind CSS**.

---

##  Live Demo  
-- **[Visit Website](https://ninjamanan.github.io)**  

---

##  Tech Stack  
- **Frontend:** HTML5, Tailwind CSS, JavaScript  
- **Effects:** Custom animations, blur overlays, sparkle trails  
- **Backend/Form Handling:** Formspree API for contact submissions  
- **Design Tools:** Blender, Unreal Engine, Figma  

---

##  Features  
- Animated entrance overlay  
- “Liquid glass” cards with subtle reflection effects  
- Responsive and mobile-friendly layout  
- Sparkle and hover interaction system  
- Contact modal powered by Formspree  
- Background video with graceful fallback  

---

##  File Structure  

```
├── index.html # Main page (everything’s in here)
├── background.mp4 # Optional video background
├── profile.jpg # Avatar / profile image
└── /assets # (Recommended) for future images, scripts, or styles
```
